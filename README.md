Hello everyone! The goal of this app is to establish a tool that can quickly intake scRNA-Seq data of interest (either as paired .mtx files with barcode and gene files for deconvolution, or .h5ad files) and quickly pull out distributions for captured genes of interest. This tool as it stands was very quickly coded by Jackson using Claude code. Jackson is not a coder and is even less of a statistician. The code is here to be reviewed by more competent coders and statisticians. A quick explanation of the codes function as Jackson understands can be found in the slides linked below:
